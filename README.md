Blinkit Sales Analysis Dashboard (Power BI)
.Overview
This project focuses on analyzing Blinkit's sales performance, customer satisfaction, and inventory distribution using Power BI.

 .Objectives
- Analyze total sales and revenue trends
- Evaluate customer ratings and satisfaction
- Understand product performance and inventory distribution
- Generate actionable business insights

 .KPIs Tracked
- Total Sales
- Average Sales
- Number of Items Sold
- Average Rating

 .Key Analysis
- Sales by Item Type
- Sales by Fat Content
- Outlet-wise Sales Performance
- Sales by Outlet Establishment

 .Tools
- Power BI
- DAX
- Data Cleaning & Transformation

 .Features
- Interactive dashboards
- Dynamic KPI tracking
- Business-driven insights
